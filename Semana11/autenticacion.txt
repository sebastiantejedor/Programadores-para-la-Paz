El login es el proceso mediante el cual un usuario se identifica en una aplicación utilizando sus credenciales para acceder a sus funciones. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar acciones importantes que no cualquier persona debería ejecutar. Esto ayuda a mantener la seguridad de la plataforma y evitar accesos no autorizados. Además, nunca deben exponerse públicamente contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves de API ni otros datos sensibles de los usuarios o de la comunidad.
