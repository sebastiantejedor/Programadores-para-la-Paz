El comando uname -a muestra información sobre el sistema operativo, como el nombre del sistema, la versión del kernel y la arquitectura del equipo.

El comando whoami muestra el nombre del usuario que actualmente está utilizando el sistema.

Conocer esta información es útil al administrar un servidor porque permite identificar el entorno de trabajo, verificar el usuario con el que se ejecutan los comandos y facilitar tareas de configuración y mantenimiento.
